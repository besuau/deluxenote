ONE PAGE USER GUIDE
English

Main Window

1. Dropdown menu you'll rarely run functions from here. Take it as a reference to refer for all available functions

2. Insert functions use the toolbar's central area icons to insert new elements. Behavior may vary according to selection

3. Properties use this panel to change the properties of inserted elements after selecting them

4. Context palette quick access to available functions for the selected elements. Useful to save time as well as to discover DeluxeNote features

5. Project browser shows the document's logic structure and let you select scores and parts to change its properties, instruments, etc.

6. Status bar shows real time information on keyboard and cursor position

Workspace

Selection

Click (left button) on an element to select it. Click on a blank area or press Esc to clear the selection

Click on the staff header (clef, key and time signatures) to select the whole staff

Click on an element and drag to move the selected elements

Click on a blank area and drag to start the window selection mode: left to right will select all kind of elements; right to left, notes only

Zoom

Use the mouse wheel to zoom in and out

Click the mouse wheel to zoom extents. Depending on the clicked area you can zoom the extents of a measure, a system, a page...

Pan

Right click and drag to pan the view

Hover over the Virtual Wheels at the bottom-right corner and use the mouse wheel to scroll

Notes

To insert notes, on the toolbar, select the note value, set the dot, rest and second voice options, and use any of the following methods:

  1. Use the mouse to place the note on the desired position and step. Click to confirm. Right click, press Esc or click on a blank area to cancel the insertion
  2. Click on any piano key. The corresponding note will be inserted at the current cursor position and the cursor will move forward. If cursor is undefined nothing will happen
  3. Press any key from "Z" to "-" positions in your keyboard. The corresponding note, based on the current keyboard octave, will be inserted as if you had clicked on the piano keyboard
  4. Press Shift + Number (1..9) to insert an interval for the selected notes
