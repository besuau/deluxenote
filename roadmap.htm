Version History

1.3.3   September 19, 2021

  • Manuscript paper (bonus tool #3)
  • Transpose Keyboard octave (Ctrl+Alt + Key/Click on Piano keyboard)
  • Keyboard octave takes now into account the keyboard language
  • Improved dragging of Dynamic, Octave and Pedal elements

1.3.2   August 1, 2021

  • Textured paper
  • Different selection color for second voice notes
  • Slightly displaces notes in different voices when opposite stems overlap or form the interval of a second/unison
  • Avoids collisions of accidentals in different voices

1.3.1   July 5, 2021

  • Playback tracking
  • Scrollbar in the Context palette when needed
  • More precise placing of Dynamic, Octave and Pedal elements

1.3   June 21, 2021

  • Score scaling
  • Staff scaling (from Part properties)
  • Time signature changes
  • Improved appearance when the Windows display scaling is higher than 100%
  • New default background color
  • Notes are now played also when inserted using the mouse
  • Search for updates is now asynchronous allowing DeluxeNote to start even faster

1.2.2   May 2, 2021

  • Transpose notes and key signatures to any key or by any interval or by semitones

1.2.1   April 5, 2021

  • Shorten measures by removing beats
  • Courtesy time signature at system end
  • Swap voices
  • Select by element type and select similar
  • Holding down the right mouse button has the same effect as holding down Shift to select a range of notes
  • Now you can insert notes directly into any existing position regardless of its voice

1.2   March 9, 2021

  • Arpeggiated chords
  • Quintuplets, septuplets, octuplets and nonuplets
  • Hide/show rests
  • Rests can now be moved vertically
  • Slur position (above/below) can now be specified or kept auto
  • Tuplet duration can now be changed in the same way as with notes
  • Tooltip messages when tool is not applicable to current selection

  • Transpose by semitones by pressing Shift + Up/Down

  • Text properties for Page numbers and System Measure numbers
  • System Measure number position can now be edited vertically
  • Accidentals are now displaced when needed to avoid collisions on chords
  • Holding down Shift while moving the cursor with Left/Right keys selects a range of Notes
  • Clicking on a voice in the Project browser selects the corresponding notes
