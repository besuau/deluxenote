DeluxeNote software privacy policy

Using DeluxeNote is private. There is no log in or sign up, no internet connection is required. The program just runs its music notation functions on the opened documents, and it only saves files at the user's request. DeluxeNote does absolutely nothing the user is not aware of.

Specially:

  • DeluxeNote DOES NOT read, modify, collect or send any data from the end-user computer.

  • DeluxeNote DOES NOT analyze user behavior, nor send usage statistics anywhere.

  • DeluxeNote uses Internet connections only to check for updates.
    • No data is sended during the check. It just request to this site the latest version number.
    • User can disable these checks from the Options dialog.
